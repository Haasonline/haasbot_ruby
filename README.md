# HaasbotRuby

This gem provides an interface to the Local API of Haasbot.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'haasbot_ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install haasbot_ruby
